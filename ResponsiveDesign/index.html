Flexbox Cheat Sheet

a Flexbox (flexible container) can be declared by setting either;
display: flex; or display: inline-flex;

The content inside a Flexbox can be manipulated many ways, focusing on either
the Main axis (Default X axis) or the Cross axis (Default Y axis)

justify-content:
manipulates content on the main axis
flex-start; > To start of flexbox
flex-end; > To end of flexbox
center; > In middle of flexbox
space-between; > split out, only space between children.
space-around; > split out, with space around children too

align items:
maipulates content on the cross axis
flex-start; > To start of flexbox
flex-end; > To end of flexbox
center; > In middle of flexbox
baseline; > bottom on the level, tops land wherever
stretch; > Default / elements will stretch (specified heights wont stretch)

flex:
shorthand for flex-grow, flex-shrink and flex-basis.
flex-grow: specifies the amount an element should grow in comparison to another
(1 is default, 2 is twice, 3 is thrice etc)
flex-shrink: specifies the amount an element should shrink in comparison to another
(1 is default, 2 is twice, 3 is thrice etc)
flex-basis: specifies the size of an element before any shrinking or growing occurs.
(500px / 750px / 1500px / etc)

flex: 1 2 500px;

flex-wrap:
declares which way (if any) the flex-box should wrap.
flex-wrap: wrap; > will move down a line.
flex-wrap: wrap-reverse; > will move up a line.
flex-wrap: no-wrap; > will not move.

align-content:
Moves MULTIPLE ROWS along the Cross Axis (Defaut Y axis)
align-content: flex-start; > child elements to the top of parent container
align-content: flex-end; > child elements to the bottom of parent container
align-content: center; > child elements to the center parent container
align-content: space-between; > child elements to the top and bottom of parent container with space between
align-content: space-around; > child elements to the top and bottom of parent container with space between, and space around.
align-content: stretch > (Default) child elements stretch to take up full height of parent row.

flex-direction:
set the start and the end of the flexbox, Left to Right? Right to left? Up to down? Down to up? any way you want it!
flex-direction: row; > (Default)By default it sets the direction to L > R ( Sets Main Axis Start To End)
flex-direction: row-reverse; > By default it sets the direction to R > L ( Sets Main Axis End to Start)
flex-direction: column; > By default it sets the direction to Top > Bottom ( Sets Cross Axis start To End)
flex-direction: column-reverse; > By default it sets the direction to Bottom > Top ( Sets Cross Axis end To Start)

Grid Cheat Sheet

Declare a grid by display: grid; or display:inline-grid;
**Grids need a specified width and height

Declare the columns by grid-template-columns: repeat(autofit, minmax(250px, 1fr))

Declare the rows by grid-template-rows: repeat(autofit, minmax(250px, 1fr))

Or Declare both with grid-template: repeat(autofit, minmax(250px, 1fr)) / repeat(autofit, minmax(250px, 1fr))

Declare a gap in rows with row-gap: x

Declare a gap in columns with column-gap: y

Declare a gap in Both with gap: x y

Grid items are the item elements used within a grid container.

set Grid Items' starting row with; grid-row-start; x

set Grid Items' ending row with; grid-row-end; y

set Both with grid-row: x y;

set Grid Items' starting column with; grid-column-start: x

set Grid Items' ending column with; grid-column-end: y

set both with grid-column: x y;

Span Keyword: used to tell the grid-item how many (Rows/Columns) to span
grid-column: 2 / span 4; start column2, span 4 columns.

Set entire grid-item's existance with grid-area; RowStart/ ColStart/ RowEnd/ ColEnd;
grid-area: 1/ 3/ span 2/ span 5;

grid areas can be also be declared with grid-template-areas:
which takes a series of strings like;

grid-template-areas: "head head"
"nav nav"
"info services"
"footer footer";

which grid-items can be allocated to using;
grid-area: "nav";
(This can also be used to overlap elemements)

Within grids -content is aimed at the entire grid as a whole, inside of its grid container.

justify-content: is used to position the entire grid along the Row Axis (X axis)
justify-content: start; > aligns grid to left side of container
justify-content: end; > aligns grid to right side of container
justify-content: center; > centers the grid horizontally
justify-content: stretch > stretches grid items to increase dize of grid on X axis.
justify-content: space-around; > creates space between and around
justify-content: space-between; > creates space between only.
justify-content: evenly; > places even amount of space between grid items at either end.

align-content: is used to position the entire grid along the Column Axis (Y Axis)
align-content: start; > aligns grid to top side of container
align-content: end; > aligns grid to bottom side of container
align-content: center; > centers the grid horizontally
align-content: stretch > stretches grid items to increase dize of grid on Y axis.
align-content: space-around; > creates space between and around
align-content: space-between; > creates space between only.
align-content: evenly; > places even amount of space between grid items at either end.

within grids -items is aimed at the content inside of each grid item

justify-items: is used to position content inside grid-items along the Row Axis (X axis)
justify-items: start; > content to left side of grid-item
justify-items: right; > content to right side of grid-item
justify-items: center; > content to center of grid-item
justify-items: stretch; > content to fill the whole grid-item

align-items: is used to position content inside grid-items along the Column Axis (Y axis)
align-items: start; > content to top side of grid-item
align-items: right; > content to bottom side of grid-item
align-items: center; > content to center of grid-item
align-items: stretch; > content to fill the whole grid-item

lastly, -self is aimed at individual grid-items.

justiy-self(X) / align-self(Y): is used to position an individual element in the same way that
multiple elements are positioned with justify-items and align-items.
justify-self: start; > left side of grid
justify-self: right; > right side of grid
justify-self: center; > center of grid
justify-self: stretch; > fill the whole grid

align-self: start; > top side of grid
align-self: right; > bottom side of grid
align-self: center; > center of grid
align-self: stretch; > fill the whole grid

implicit grid rules take over when the grid becomes larger than what is set out by the explicit grid rules.
grid-auto-rows: is the implicit counterpart to grid-template-rows.
grid-auto-rows: 50px; > sets to 50px
grid-auto-rows: 1fr; > sets to 1 fraction;

grid-auto-columns: is the implicit counterpart to grid-template-columns.
grid-auto-columns: 50px; > sets to 50px
grid-auto-columns: 1fr; > sets to 1 fraction;

grid-auto-folow: sets where new elements will be added
grid-auto-flow: row; > (Default) sets new elements fill rows from L to R. Create new Rows.
grid-auto-flow: column; > sets new elements to fill columns from Top to Bottom. Create New Cols.
grid-auto-flow: dense; > sets new elements to fill existing holes in grid.

dense can be paired with row and column.

Responsive Design Projects

  • Project 24

In a nutshell, using fixed pixel values is a bad idea for making a responsive website.
use dynamic values like rem (Root em) and % (Percentage)
(root font size is 16px)
font-size: 1rem; > 16px;
font-size: 2rem; > 32px;
While math is hard and all, we can amend the default font-size to 10px, and hey presto, using rem is cake.

width: 50%; > 50% of its parent
height: 75%; > 75% of its parent.

Limiting sizes is a great way to set an upper and lower bound
max-width: 500px;
min-width: 250px;
These rules ensure that the image width will be within 250 and 500px.

We can take this a step further with clamp();
which takes 3 arguements, minimum value, preferred value, and maximum value.
font-size: clamp(1rem, 2.5vw, 2rem);
The above sets the font-size to between 1 and 2 rem.

Scaling images/Videos
Set a container to have a specific value, then set the content within said container to be max-width 100% and dynamic

.container {
width: 50%;
height: 200px;
overflow: hidden;
}

.container img {
max-width: 100%;
height: auto;
display: block;
}

Scaling background Images; set background image to be scalable by

.background {
background-image: url('#');
background-repeat: no-repeat;
background-position: center;
background-size: cover;

}

Media queries allow CSS to inspect the device viewing the media,
then show specific media based on the device queries.
The only requrement for this is for the HTML to have a meta tag with viewport.
<meta name="viewport" content="width=device-width, initial-scale=1.0" />

Media Query Examples:
@media (condition) {

}

AND keyword requires both/all conditions to be satisfied:
@media only screen and (max-width: 480px) {
>only for screens
>with a max-width of 480px
}

@media only screen and (min-width: 320px) and (max-width: 480px) {
>only for screens
>with a min-width of 320px
>with a max-width of 480px
}

>>Useful to serve higher quality images
@media only screen and (min-resolution: 300dpi) {
>only for screens
>with a min-resolution of 300dpi
}

comma seperated list requires either or condition to be satisfied:
@media only screen and (min-width: 480px), (orientation: landscape) {
>only screen
>and min-width 480px // and orientation landscape.
}
