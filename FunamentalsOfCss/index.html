Box Model

Content > Padding > Border > Margin

Traditionally, Width and Height consist of only the content... the rest is extra.

With box-sizing: border-box; the whole box model is represented within the Width and Height.

width declares the width of the element
height declaresd the height of the element

border specified the border of an element ~ Size / Style / Color
border-radius specifies the amount of curvature on the corners of the border

padding defines the amount of space beteen an element and its border
padding: 10px ~ Padding on all 4 sides
padding: 10px 20px ~ 10px on Yaxis, 20px on Xaxis
padding: 10px 20px, 30px ~ 10px on top, 20px on Xaxis, 30px on bottom
padding: 10px 15px, 20px, 30px ~ Top / Right / Bottom / Left

We can use padding-top, padding-right, padding-bottom, padding-left to set the individual values too

margin defines the amount of space between an element's border and another element's border.
margin: 10px ~ Margin on all 4 sides
margin: 10px 20px ~ 10px on Yaxis, 20px on Xaxis
margin: 10px 20px, 30px ~ 10px on top, 20px on Xaxis, 30px on bottom
margin: 10px 15px, 20px, 30px ~ Top / Right / Bottom / Left

We can use margin-top, margin-right, margin-bottom, margin-left to set margin individual values too


To automatically center an element, specify a width, then set margin to 0 auto;

elements obey the rule of "margin collapse". Margins will share space where they can.
Element A's margin-left is 20px. element B's margin-right is 30px.
The total margin will be 30px. they will share.

Layout

position defines the type of positioing an elemement has.
position is used in conjunction with 2 of the 4;

top ~ move the element down from the top by X
bottom ~ move the element up from the bottom by X
left ~ move the element right from the left by X
right ~ move the element left from the right by X

position: absolute; (Moved 100px to the left) allows us to move an element RELATIVE TO ITS DEFAULT location

position: absolute; allows the element to be ignored by all other elements

position: fixed; allows the element to be fixed to the screen (Will Not Move)

position: sticky; allows an element to stick to the viewport

position: sticky; allows an element to stick to the parent element
